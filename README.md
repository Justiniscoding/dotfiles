# MacOS Dotfiles
My MacOS dots for some of the tools I use the most. They are all completely free and open source.
- [Aerospace](https://github.com/nikitabobko/AeroSpace) - MacOS tiling window manager that is similar to i3
- [SketchyBar](https://github.com/FelixKratz/SketchyBar) - Custom menu bar for MacOS configurable with bash or lua
- [neovim](https://github.com/neovim/neovim) - A modern fork of vim
- [skhd](https://github.com/koekeishiya/skhd) - A fast and reliable hotkey daemon
- [Ghostty](https://github.com/ghostty-org/ghostty) - Customizable cross-platform terminal emulator with GPU acceleration
